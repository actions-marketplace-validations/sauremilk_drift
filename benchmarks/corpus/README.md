# MyApp

A web application for order management.

## Project Structure

- `src/myapp/handlers/` — Request handlers
- `src/myapp/models/` — Data models
- `src/myapp/utils/` — Utility functions
- `src/myapp/connectors/` — Database connectors
- `src/myapp/views/` — View templates (DIA target: does not exist)
- `src/myapp/middleware/` — Middleware layer (DIA target: does not exist)
- `tests/` — Test suite

## Getting Started

Install dependencies and run the application.
